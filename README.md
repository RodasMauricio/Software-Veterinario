## Aplicación de Gestión para Veterinarias
<p>
Aplicación de escritorio desarrollada en Windows Forms (.NET 8.0) utilizando C# como lenguaje principal. Implementa una arquitectura en capas, además de manejo de excepciones y validaciones.
</p>

### Funcionalidades principales
- Clientes, Pacientes, Veterinarios y Servicios
	- Alta, modificación y eliminación.
	- Listado de registros activos y recuperación de elementos eliminados.

- Turnos
	- Agendar, modificar y eliminar turnos.
	- Listar turnos programados.
	- Búsqueda por fecha o por paciente.

- Historial clínico
	- Registro de consultas y evolución de cada paciente.

- Persistencia de datos
	- Toda la información se almacena en una base de datos SQL Server Express incluida en el proyecto.


#### Tecnologías utilizadas
- C#
- .NET 8.0
- Windows Forms App
- SQL Server Express



#### Entorno de desarrollo
Este proyecto fue desarrollado utilizando **Microsoft Visual Studio 2022**.  
Se recomienda usar Visual Studio para abrir y compilar la solución (`.sln`), ya que facilita el manejo de proyectos .NET y el uso del diseñador visual.
